/conversationに遷移後にマイクを使って音声認識を開始してください
  - 音声認識にはWeb Speech Apiを利用します
  - 画面表示とともにWeb Speech ApiをStartしてください
  - 言語は日本語を指定してください
  - 読み取った文章はリアルタイムに画面に表示してください
    - 「ここにAI問診の会話インターフェースが表示されます。」を非表示にしてください
    - 同じ場所に読み取った文章を表示します
  - 読み取った文章の下にボタンを表示してください
    - 電話のアイコンを赤色で表示してください
    - アイコンはfontAwesomeを使ってください
    - ボタンをクリックしたらWeb Speech Apiをendしてください
